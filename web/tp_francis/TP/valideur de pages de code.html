Tester la validité de son code

Vous devez toujours vous efforcer d’écrire un code valide. Cela évitera des bugs potentiels et votre site sera au final mieux référencé sur les moteurs de recherche. Pour vérifier la validité d’un code HTML ou CSS, le w3c (World Wide Web Consortium), c’est-à-dire l’organisme qui gère l’évolution des langages comme le HTML et le CSS entre autres, a mis à disposition des validateurs de code, gratuits. Vous pouvez trouver les validateurs HTML et CSS aux adresses suivantes :

  • cliquez ici pour Validateur code HTML chez W3c

  • cliquez ici pour Validateur votre code CSS chez W3c
